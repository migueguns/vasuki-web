# vasuki-web

Landing del producto **Vasuki**: vaso con hielo sellado al vacío.

## Desarrollo

```bash
npm install
npm run dev
```

Compilar para producción: `npm run build` · Vista previa: `npm run preview`

## Lista de espera

El formulario usa `mailto:` (sin base de datos). Sustituye `hola@vasuki.example` en `src/components/CTA.tsx` por tu email real.
